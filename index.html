phat dang hoc

nhánh main

* Tạo nhánh mới trên máy tính

"git branch -c ten_nhanh"

* Chuyển nhánh

"git checkout ten_nhanh"

* Push code từ nhánh con lên github

"git push -u origin ten_nhanh"
"git push --set-upstream origin ten_nhanh"

Tham số "-u" để viết tắt cho tham số --set-upstream: thường sử dụng cho lần đầu tiên
Git sẽ tự động thiết lập một mối quan hệ theo dõi giữa nhánh cục bộ (local branch)
và nhánh từ xa (remot branch).
Trong trường hợp này git tạo liên kết giữa nhánh phatdanghoc trên local và nhánh phatdanghoc
trên máy chủ git. Từ lần 2, đứng tại nhánh phatdanghoc chỉ cần dùng lệnh "git push"
